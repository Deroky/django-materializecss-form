# materialize-css-form
Materializecss for Django Form

A simple Django template tag to work with Materializecss



## Install

```
pip install materializedjangoform
```

Add to INSTALLED_APPS:

*myapp/settings.py*

INSTALLED_APPS = [
     'bootstrapform',
     # ...
]

## Usage

Use it like this, simple.

{% load materializecss %}
{{ form|materializecss }}
